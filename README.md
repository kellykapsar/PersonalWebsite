# PersonalWebsite
Personal website developed through Hugo and RStudio Blogdown and hosted by Netlify. 

Made using a modified version of this [tutorial](https://www.youtube.com/watch?v=9Jqvaoeh1W4) from CradleToGraveR on Youtube. 

More info on how to customize the Academic template [here](https://wowchemy.com/docs/).
Fun [tutorial](https://annielyu.com/2020/01/12/blogdown-website/) by Annie Lyu on how to customize the Academic templat with step-by-step instructions.
